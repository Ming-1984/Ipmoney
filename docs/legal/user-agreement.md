# 《用户协议》（P0 口径，占位，待法务确认）

> 适用于平台所有用户（买家/卖家/机构/技术经理人）。用于账号注册、使用平台服务与交易行为的基本约束。

## 1. 服务范围
- 平台提供信息展示、检索、撮合与交易流程管理服务，不作为交易标的权属保证方。

## 2. 账号与安全
- 用户需提供真实、准确、完整的信息并及时更新。
- 账号仅限本人使用，禁止出租、出借、转让。

## 3. 用户行为规范
- 禁止发布虚假/侵权/导流信息，禁止绕过平台进行交易。
- 禁止恶意刷量、骚扰、辱骂、传播违法违规内容。

## 4. 知识产权与权属承诺
- 发布交易信息的用户需保证其具备合法权属或合法授权。
- 若存在质押/许可等权利限制，须如实披露。

## 5. 责任限制
- 平台为居间服务方，交易风险由交易双方根据合同自行承担。
- 平台在法律允许范围内不对间接损失承担责任。

## 6. 争议处理
- 争议由平台基于证据进行协调处理；必要时可引导至仲裁或司法途径。

## 7. 协议更新与通知
- 协议更新将以公告/站内信等方式提示，继续使用视为同意。
