# 《专利交易居间服务协议》（P0 口径，占位，待法务确认）

> 说明：平台提供居间撮合服务，不直接参与交易标的权属转移。

## 1. 服务内容
- 信息发布、检索、撮合与订单流程管理。
- 订金/尾款托管与状态推进。

## 2. 费用与结算
- 平台服务费/佣金由卖家承担（见《交易与结算规则》）。
- 发票仅针对平台服务费/佣金开具。

## 3. 双方责任
- 买卖双方对交易标的真实性、合法性负责。
- 线下签约与权属变更由双方履行。

## 4. 风险提示
- 平台不对权属纠纷/合同履行风险承担担保责任。
- 平台有权基于风控与合规暂停/下架。

## 5. 争议处理
- 平台可基于证据进行协调处理，必要时引导仲裁/司法途径。
