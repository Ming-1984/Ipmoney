# 术语表（统一口径）

> 目标：同一概念在用户端/后台/文档中使用同一术语，避免“同义不同名”导致理解成本与实现偏差。

## 1. 交易与订单

- **订金**：用于锁定交易并启动跟单流程（避免使用“定金”）。
- **尾款**：除订金外的剩余款项；在平台内托管支付。
- **托管**：款项由平台/支付渠道托管，满足放款条件后再结算给卖家。
- **放款/结算**：满足放款条件后，平台确认出账并归档凭证。
- **退款**：订金/尾款的退回；需符合规则与证据材料要求。
- **里程碑**：合同签署/材料补齐/提交变更/变更完成等关键节点。

## 2. 内容与上架

- **上架**：专利交易信息/需求/成果在前台可被检索与浏览的状态。
- **书画专区**：书法/绘画作品的交易专区，交易规则与专利交易一致（订金 + 尾款）。
- **书画成果**：数据地图与统计口径中的书画作品集合（用于地图/分析，不等同于交易入口）。
- **审核**：后台对内容/身份资料的审核流程（状态：待审核/已通过/已驳回）。
- **驳回**：审核不通过的处理结果；需填写原因并对发布者可见（按权限）。

## 3. 用户与身份

- **身份选择**：新用户首次进入选择主体类型（个人/企业/科研院校/政府/行业协会学会/技术经理人）。
- **认证**：提交主体资料用于审核（用于“可交易/可咨询”等权限判断）。
- **审核通过**：允许进行收藏/咨询/下单/支付等动作（按产品策略）。

## 4. 书画与服务

- **著作权登记证书编号**：书画作品权属核验的核心编号字段；P0 先人工核验，P1 可接入外部检索源。
- **技术经理人栏目**：展示审核通过的技术经理人名单，支持检索与在线咨询。

## 5. 行为与沟通

- **收藏**：用户对内容的收藏行为（用于“我的收藏/热度”等）。
- **咨询**：用户发起的站内沟通入口；默认走站内消息留痕。
- **消息/会话**：咨询产生的沟通记录；用于证据留痕与纠纷处理。

## 6. 文案禁用词

用户可见区域避免出现：`P0/演示/Mock/undefined/HTTP xxx/堆栈` 等技术词或调试信息。
