# WeApp 手工冒烟清单（P0，持续会话版）

> 目标：在接入真实微信登录/支付前，验证小程序核心链路可用，且客服/投诉已统一为“持续会话”。
> 适用环境：`scripts/start-dev.ps1 -EnableDemoAuth`（本地 API + Demo Auth）。

## 1. 前置条件

- 已启动开发环境：
  - `powershell -ExecutionPolicy Bypass -File scripts/start-dev.ps1 -EnableDemoAuth`
- API 健康检查通过：
  - `http://127.0.0.1:3200/health` 返回 `ok=true`
- 微信开发者工具已打开 `apps/client` 项目。

## 2. 冒烟用例（必测）

### A. 首页与搜索

- [ ] 首页首屏可渲染，无白屏、无明显报错。
- [ ] 首页快捷入口可跳转（至少 1 条）。
- [ ] 搜索页输入关键词后可返回结果。

### B. 详情与咨询

- [ ] 挂牌详情可打开，点击“咨询”可创建/进入会话。
- [ ] 技术经理详情可打开，咨询入口可用。

### C. 客服中心（持续会话）

- [ ] 进入 `/subpackages/support/index`，点击“在线客服会话”可进入聊天页。
- [ ] 在“意见反馈”填写内容提交后：
  - 自动创建/复用 `support` 会话；
  - 反馈内容作为消息写入会话；
  - 自动跳转聊天页。
- [ ] `/subpackages/support/contact/index` 中“在线客服会话”可正常进入会话。

### D. 订单争议（投诉）

- [ ] 进入订单详情 `/subpackages/orders/detail/index`。
- [ ] 点击“争议沟通”后可创建/复用 `dispute` 会话并进入聊天页。
- [ ] 聊天页上下文卡片可回跳订单详情。

### E. 聊天连续对话

- [ ] 聊天页可发送文本消息。
- [ ] 发送失败时可点击“重试”。
- [ ] 上拉/点击“加载更早消息”可获取历史消息。
- [ ] 返回消息页后可看到对应会话与最新预览。

### F. 后台客服工作台联动

- [ ] 管理后台 `/conversations/platform` 可看到咨询/客服/争议三类会话。
- [ ] 按 `channel`、`assigned`、`q` 筛选生效。
- [ ] 可执行“分配给我/指定分配/移除坐席”。
- [ ] 分配后可持续回复并刷新会话。

### G. 年费托管链路（小程序 x 后台）

- [ ] 小程序“我的”可进入 `/subpackages/maintenance/index`，能看到“年费计划/托管任务”双视图。
- [ ] 计划/任务筛选可用，分页加载与下拉刷新正常。
- [ ] 对“7天内到期/逾期”记录点击“联系客服”可进入持续会话聊天页。
- [ ] 管理后台 `/maintenance` 可按状态/ID/日期筛选计划与任务。
- [ ] 管理后台任务编辑支持证据文件上传并自动回填 `evidenceFileId`。
