# 管理后台首页运营配置低代码化（2026-04-14）

## 改造目标
- 把原先纯 JSON 编辑页升级为“可视化低代码 + JSON 高级模式”双编辑器，降低运营使用门槛。

## 已上线能力
1. 可视化编辑（低代码）
- Hero 标签与搜索占位文案。
- 特色专区标题/更多文案。
- 展示开关与展示数量（4/6）。
- `listingTopicUi` 的名称、排序、可见开关（全链路同步）。
- 特色卡片的增删改、排序、动作配置（搜索预填 / 页面路由）。

2. JSON 高级模式
- 支持批量粘贴 JSON、应用到可视化、直接保存。
- 保存前统一做结构标准化。

3. 前端校验
- 特色专区启用时，启用卡片数必须 >= 展示数量。
- 卡片标题/副标题/图片地址必填。
- `PAGE_ROUTE` 必填 URL。
- `SEARCH_PREFILL` 绑定的 `listingTopic` 不能是已关闭标签。
