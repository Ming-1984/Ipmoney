# 管理后台统一上传与品牌展示对齐说明（2026-04-16）

## 目标
- 统一管理后台图片/文件上传交互，降低运营配置成本。
- 平台统一发布场景下，对外展示名统一为 `ipmoney`，避免暴露个人账号昵称。

## 一、管理后台统一上传逻辑
统一入口组件：`apps/admin-web/src/ui/ImageUrlUploadField.tsx`

统一链路：
1. 页面点击上传按钮，选择图片文件。
2. 前端调用 `POST /files`（`apiUploadFile`）。
3. 后端返回 `FileObject`（`id / url / mimeType / sizeBytes`）。
4. 表单自动回填可访问 `url` 并支持预览。
5. 保存业务配置后，小程序/后台页面按配置实时生效。

说明：
- 运营侧可“上传即用”，不再要求手工粘贴对象存储 URL。
- 仍兼容直接填写 URL（用于历史数据或外部素材）。

## 二、已统一到上传组件的页面
- 首页运营配置（特色专区图片等）
- Banner 相关图片位
- 技术经理人头像
- 认证审核 Logo

## 三、咨询导入后台后的账号显示规则
- “咨询人”始终显示真实买方账号（用于客服识别与跟进）。
- 平台方（卖方/平台接待方）在“平台统一发布 + 平台路由”场景下，对外统一显示 `ipmoney`。
- 登录账号、权限主体、审计日志保持真实账号，不做篡改。

## 四、本次展示名统一规则（已落地）
- 挂牌详情：`source=ADMIN/PLATFORM` 且 `consultationRouting=PLATFORM` 时，供方昵称统一返回 `ipmoney`。

- 平台会话中心：咨询人保持真实买方昵称，不覆盖为品牌名。
